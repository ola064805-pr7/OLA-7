
# OLGA DELF — Творческий Альманах

Персональный сайт-портфолио Ольги Храпутской.

## О проекте
Сайт выполнен в стиле **Newsprint** (газетная верстка). 
Разделы включают:
- **Music for Gymnastics Performances** (AI музыка для спорта)
- **Сказки из мира фантазий** (Авторская литература)
- **Эвритмия** (Движение и жест)
- **Магия Нейрофото** (Искусство ИИ)

## Технологии
- React 19
- Tailwind CSS (Стилизация)
- Vite (Сборка)
- Lucide React (Иконки)

## Развертывание
Сайт автоматически деплоится на **Vercel** при пуше в ветку main.
